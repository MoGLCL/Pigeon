# Pigeon Release Sources

This folder is generated by the packaging process of the main development repository. These variants represent independent source distributions tailored for specific environments.

For issues, contributions, or building from source, visit the **[Pigeon Main Repository Hub](https://github.com/MoGLCL/Pigeon)**.

## Release Packages

- **`Pigeon-Windows-Local`**: Local Windows source without Docker files.
- **`Pigeon-Linux-Docker`**: Linux source with Dockerfile and Docker Compose.
- **`Pigeon-Private-Hosting`**: Node/VPS/control-panel source with PM2 and production templates.
- **`Docs`**: Standalone deployment documentation handbook and Markdown guides.

No live `.env`, secrets, database files, build outputs, `node_modules`, or runtime logs are included. All packages are clean and production-ready.
